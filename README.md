# 📚 BookhuB – A Hub for Better Learning

BookhuB is a student-focused web application designed to provide semester-wise study materials, academic resources, and useful tools for BCA students in one place.

The main goal of this project is to make learning more organized, accessible, and convenient.

---

## ✨ Features

- 📖 Semester-wise Notes with One-Click Download
- 🗺️ Structured Web Development Learning Roadmap
- 📊 CGPA / SPI to Percentage Calculator
- 📱 Fully Responsive Design
- 🌙 Dark Mode Support
- 💬 Student Feedback Form

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---



## 🎯 Project Objective

Many students struggle to find notes and academic resources from different sources.

BookhuB solves this problem by providing a centralized platform where students can easily access study materials and useful academic tools.
